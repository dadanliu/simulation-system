
```text

  |
  v

  |
  v

  |

  |
  v
```


新增命令：

```bash

pnpm test:client:e2e:headed
```

两者区别：

```text
test:client:e2e
  -> 无头浏览器，适合 CI

test:client:e2e:headed
  -> 打开真实浏览器窗口，适合本地调试
```

## 18.6 Prettier 和 ESLint 配置

Prettier 改为更适合稳定 diff 的换行策略。

相关文件：

```text
prettier.config.mjs
.vscode/settings.json
```

配置：

```text
printWidth: 80
editor.rulers: [80]
```

目的：

```text
更早换行
  |
  v
后续新增参数或 import 时
不容易从一长行突然变成多行
  |
  v
diff 更稳定
```

ESLint 忽略 Playwright 产物：

```text
eslint.config.mjs
.gitignore
```

忽略目录：

```text
playwright-report/**
test-results/**
```

原因：

```text
这些是 Playwright 生成的报告、trace、截图、视频
不是项目源码
不应该被 ESLint 扫描
```
